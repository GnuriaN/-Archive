# PowerShell
Scripts in PowerShell

## Get-HwInfo.ps1
> Скрипт для получения информации о системе с использованием WIN32 API (http://msdn.microsoft.com/en-us/library/aa389273(VS.85).aspx). 
>> В настоящий момент
+ реализовано:
  + Operation System
  + MatherBoard
  + BIOS
  + Processor
  + Random Access Memory
  + Video Adapter
  + Disk media
  + Logical disk
  + Network Adapter
+ не реализовано:
  + CD Rom
  + Printer
  + Other devices
  
  ## ...
